비빔밥

비빔은 한국의 밥 요리이다. 쌀밥에 고기나 나물 등과 여러 가지 양념을 넣어 비벼 먹는 음식이며, 부빔밥이나 제삿밥, 골동반, 교반으로도 부른다. 과거 궁중에서는 비빔이라 불렀다. 대한민국의 국민 음식 가운데 하나로 여겨진다.
